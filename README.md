<h2 style="color:#1e3d59;">Como Usar (Desenvolvedores)</h2>
<ol style="line-height:1.8; color:#34495e;">
  <li>Clone o repositório: <code>git clone https://github.com/eronaldo-coelho/Boleo.git</code></li>
  <li>Entre na pasta do projeto: <code>cd Boleo</code></li>
  <li>Instale as dependências Flutter: <code>flutter pub get</code></li>
  <li>Execute no seu dispositivo: <code>flutter run</code></li>
</ol>
